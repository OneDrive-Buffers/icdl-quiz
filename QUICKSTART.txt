🚀 TO GET STARTED

Option 1: Quick Browser Launch (Easiest!)
  1. Find "index.html" in this folder
  2. Double-click it
  3. Start taking the quiz!

Option 2: Using Python Web Server
  1. Open Command Prompt/Terminal in this folder
  2. Type: python -m http.server 8000
  3. Open browser to: http://localhost:8000
  4. Click index.html

Option 3: Using Node.js
  1. Open Command Prompt/Terminal in this folder
  2. Type: npx http-server
  3. Follow the displayed URL

🔧 TROUBLESHOOTING

  Problem: Browser won't load the page
  Solution: Make sure you have index.html selected, and try a
           different browser (Chrome, Firefox, Edge, Safari)

  Problem: Questions not showing
  Solution: Verify the "questions" folder exists in this directory
           with JSON files inside

  Problem: Want to add more questions
  Solution: Edit the JSON files in the questions/ folder following
           the same format as existing questions
